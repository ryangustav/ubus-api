# External Integrations

**Analysis Date:** 2026-03-30

## APIs & External Services

**Email/SMS:**
- SMTP (Nodemailer) - Transactional emails (registration, password resets)
  - SDK/Client: `nodemailer` npm package v6.9.16+
  - Auth: Credentials (host, port, user, pass) in `SMTP_*` env vars
  - Templates: Local integration in `src/shared/email`

**Cloud Providers:**
- Oracle Cloud Infrastructure (OCI) - Secrets management
  - SDK/Client: `oci-secrets` / `oci-common` v2.126.1
  - Auth: Instance Principals (for production)
  - Services: OCI Vault (OCID in `OCI_SECRET_OCID` env var)

## Data Storage

**Databases:**
- PostgreSQL - Primary data store
  - Connection: via `DATABASE_URL` env var
  - Client: Drizzle ORM (via `pg` driver)
  - Migrations: Managed by `drizzle-kit` in `./drizzle`

**Caching & Queues:**
- Redis - Session caching and BullMQ job management
  - Connection: `REDIS_URL`, `REDIS_HOST`, `REDIS_PORT`, `REDIS_PASSWORD` env vars
  - Client: `ioredis` v2.0.0 (NestJS module)
  - Usage: Tokens, BullMQ task management

## Authentication & Identity

**Auth Provider:**
- Custom JWT (Passport) - Strategy-based authentication
  - SDK/Client: `@nestjs/jwt`, `@nestjs/passport`
  - Implementation: `JwtStrategy` in `src/modules/auth/application/strategies`
  - Token Management: `JWT_SECRET` for signing, TTL managed in app

## CI/CD & Deployment

**Hosting:**
- Oracle Cloud Infrastructure (OCI) - Ampere (ARM64) Instances
  - Deployment: GitHub Actions + Docker Compose
  - Networking: Nginx Reverse Proxy (Port 80 -> 3001)

**CI Pipeline:**
- GitHub Actions - Automated deployment
  - Workflows: `.github/workflows/deploy.yml`
  - Lifecycle: Build image -> Push to OCI Registry -> Deploy via SSH to VPS

## Environment Configuration

**Development:**
- Local `.env` file
- Local PostgreSQL and Redis via Docker Compose
- `USE_OCI_VAULT` set to `false`

**Production:**
- `.env` generated by GitHub Actions (mounted RO)
- `USE_OCI_VAULT` set to `true` (fetched at runtime)
- Secrets management: OCI Vault + GitHub Secrets
